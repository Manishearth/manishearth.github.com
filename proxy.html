Manishearth

This script lets one quickly control most of the proxy settings on Ubuntu, including the APT (Ubuntu Software Center) settings and browser settings without having to know anything about Ubuntu. It also helps one install chrome/chromium and simplecpp. In addition to initial setup, it also lets one reconfigure the settings with new userids or by turning them off altogether.

Note that I have not thoroughly tested this yet, but everything it does is reversible. Drop me a mail at manishg@iitb.ac.in if you have any issues/suggestions.

Instructions:
  1. Download script from here. If the script doesn't directly download, hit Ctrl-S, navigate to your Downloads directory, and save it there. For convenience, let it stay in your Downloads directory
  2. Open Terminal (Ctrl-Alt-T)
  3. Navigate to the script directory. If you have saved it in your Downloads folder, simply type cd ~/Downloads and hit Enter.
  4. Type bash proxy.sh and hit Enter. The script should start. It will ask you for your Ubuntu password as well as your LDAP login info.
  5. After this, one should get a menu. For most purposes, choosing option 1 should be enough, it walks through the entire setup, asking you for confirmation whenever there is an optional step.
  6. The rest will be explained by the script :)
